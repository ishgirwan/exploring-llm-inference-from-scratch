# Failures

Wrong turns, recorded as they happen — not reconstructed later. The clean
final code is the least interesting thing in this repo; this file is where
the learning actually shows. Newest entries first.

Each entry follows one shape:

```text
## YYYY-MM-DD — one-line name of the wrong turn
what I expected · what actually happened · what it cost ·
the fix, or the question it left open
```
